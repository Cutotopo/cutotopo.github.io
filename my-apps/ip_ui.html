# User interface

InkPaint's UI is really easy to use.

# Start screen

  1. New file/Open file: You can use those buttons to create a new file or to open one.

# "Pen" menu

  1. Canvas: Here you can draw with the pen
  2. Pen thickness: select the thickness of the pen, from 0,5 to 6,5
  3. Default thickness: resets the default thickness (1,5)
  4. Pen color: you can choose between black, blue, red, yellow, green, white
  5. Tools: the program supports 5 modes: Draw, Select, Rubber, Delete stroke, Read only

# "Tools" menu

  1. Highlight: enables or disables the highlighter

# "View" menu

  1. Zoom: zooms in or out on the strokes
  2. Guides: you can add rows or squares under your strokes
  3. Full screen (broken)

# "Background" menu

  1. Load picture: loads a *.png or *.jpg file as a background
  2. Remove picture: removes the background picture

# Backstage menu

  1. "about this file": provides general information about the file
  2. "new canvas": deletes every stroke in the canvas
  3. "open": loads the canvas from a stroke
  4. "save": saves the file
  5. "save as": saves the file with a different name or a different format
  6. "highlighter": enable or disable the highlighter
  7. "input": select which input is allowed to draw
  8. "background": add or remove a background image
  9. "print": prints the canvas
  10. "close": closes the application and opens the start screen
